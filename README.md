# ELO-based-pingpong
Round-robin doubles ping pong tracker with Elo + stats (Python)

Features include in this programme:
- randomized teams based on 6 players
- score tracking of each match
- point allocation system to each player
- final ranking assignment
- ELO recalibration
- ELO based point allocation i.e. vs HIGH ELO = more points, vs LOW ELO = less points
